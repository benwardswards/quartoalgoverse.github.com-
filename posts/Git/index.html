Git commands

Initiating a git repository from a folder:

git init

Adding and Removing Files From the Index

Checking which file differs from current working director and the head:

git status
git rm --cached

Find the list of files in the index

git ls-files

Checking History of Commits

List of commits

git log

List commits with oneline and in graph mode to see branches,

git log --graph --oneline

List of all recent git commits including amends and reverts

git reflog

Managing Branches

View all exists branches

git branch

How to delete a branch

git branch -D <branch_to_delete>
git  rebase

SSH Keys and Passphrases with Github

  • Generate a private public key pair
    • The public key in ~/.ssh is .pub file
  • Get that key to run with ssh
  • Upload the public key to github
  • Set the remote with ssh
ssh-add ~/.ssh/id_ed25519
ssh-keygen -t ed25519 -C "wardsbd@gmail.com"
eval "$(ssh-agent -s)"

The command line authenticates with ssh so you have to set the remote with ssh not https. Go to your github page like you were going to find the clone address and choose ssh to down load the correct link.

git remote set-url origin git@github.com:benwardswards/quartoalgoverse.github.io.git

Checking your credentials:

ssh -T git@github.com
